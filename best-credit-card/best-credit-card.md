## 1位：外為どっとコムカード
### 初年度3%、2年目以降も2%還元で外貨貯金
年会費無料の[外為どっとコムカード](https://www.gaitame.com/service/rakutsumu/creditcard/)は**初年度3%、2年目以降も2%還元**と、年会費無料カードの中でもトップクラスの高還元。

※月30万円を超えた利用分は1%還元になる点は注意が必要です。

このカードで貯まるFXポイントは、[外貨積立（らくつむ）](https://www.gaitame.com/service/rakutsumu/)というアプリで外貨と交換できます。

たとえば、**1,000FXポイント → 1,000円分の米ドル**に交換可能。

さらに、外為どっとコムで外貨を保有していると、**スワップポイント（利息）**も受け取れます。

（スワップカレンダーの画像）

米ドルの場合、2025年9月のスワップポイントは年利およそ3.6%（※為替状況により変動）。

10万円分の米ドルを保有していると、
```
100,000円 × 0.036 ÷ 365日 ≒ 1日あたり約10円
```
の利息がもらえます。

## 2位：PayPayカード
### 最大1.5%還元。ウェル活が使えれば実質2.25%（予定）
年会費無料の[PayPayカード](https://www.paypay-card.co.jp/)は、**条件達成で最大1.5%還元**。

2025年5月に発表された[PayPayポイントとVポイントの相互交換](https://about.paypay.ne.jp/pr/20250515/01)が実現すれば、ポイントを[ウェル活](https://media.yourmystar.jp/articles/11622bb0-3e6d-45d8-8f68-5843368195d3)で使えるようになります。

ウェル活ではポイント価値が**1.5倍**になるため、
```
1.5% × 1.5倍 = 実質2.25%還元
```
という計算です。

ただし、1.5%還元はPayPayステップ達成の翌月のみ。未達成の場合は1%還元になります。

### PayPayステップの条件
毎月1日〜月末までの1ヶ月で、
- 200円以上の支払いを30回
- 合計10万円以上の支払い

が必要です。

Amazonギフト券のチャージなどで達成する方法もありますが、利用規約には以下の記載があります。

> 通常1回の決済にて支払うべき商品等代金を、複数回に分割する決済により特典付与を受けた場合は対象外

そのため、
- Amazonギフトの毎週チャージ
- モバイルPASMOへのチャージ
- ANAPayが使えないネット決済
- 家族カードの利用

など、**複数の支払い方法を組み合わせて達成するのがおすすめ**です。

なお、PayPayカードでワイモバイル料金を支払うと[毎月330円割引](https://www.ymobile.jp/sp/paypay-card/)になります。

## 3位：PayPayカード ゴールド
### 最大2%還元。年会費11,000円
年会費11,000円の[PayPayカード ゴールド](https://paypay.ne.jp/card-gold/)は、条件達成で**最大2%還元**。

通常のPayPayカードとの主な違いは、
- 年会費：11,000円
- 還元率：＋0.5%

そのほか、
- 空港ラウンジ無料
- LYPプレミアム無料
- ワイモバイル料金が毎月550円引き

といった特典もあります。

年会費11,000円を、還元率アップ分（0.5%）で回収するには**年間220万円以上の利用**が目安。

つまり、**年間220万円以上使うなら、通常のPayPayカードよりゴールドの方がお得**です。

## 4位：リクルートカード
### 無条件で1.2%還元。とにかくシンプル
年会費無料の[リクルートカード](https://recruit-card.jp/)は、**条件なしで1.2%還元**。

利用上限もなく、管理がとてもラクなのが魅力です。

貯まったリクルートポイントは、**dポイント・Pontaポイントへの交換やAmazonでの利用**が可能。

年に数回行われる**dポイント10%増量キャンペーン**を活用すれば、

```
1.2% × 1.1倍 = 実質1.32%還元
```
になります。

（dポイント増量の画像）

「ポイント条件を気にするのが面倒」という人にリクルートカードは向いています。

リクルートカードは[最大6,000ポイント還元キャンペーン](https://recruit-card.jp/campaign/newadmission/)を実施中です。

## 5位：P-oneカード＜Standard＞
### 1%自動キャッシュバックで管理ゼロ
年会費無料の[P-oneカード＜Standard＞](https://www.pocketcard.co.jp/card/card_pone_s.html)は、利用額の**1%が自動で値引き**される珍しいカード。

ポイント管理・有効期限・交換先を気にする必要がありません。ポイント管理が面倒なシンプル派にはかなりおすすめです。

P-oneカード＜Standard＞は**2026年4月30日まで[最大15,000円還元キャンペーン](https://www.pocketcard.co.jp/ad/pone_s/nm-nor/)**を実施中。

## クレカはプリペイドカードにチャージしてポイント多重取り
クレジットカードは、プリペイドカードにチャージして使えば**ポイントの多重取り**が可能です。

### Mastercard / JCBの場合
```
クレカ → JALPay(0.1%)
→ ANAPay(0.5%)
→ 楽天Pay(1.5%)
```
→ **＋2.1%還元**

### VISAの場合
```
クレカ → ANAPay(0.5%)
→ 楽天Pay(1.5%)
```
→ **＋2%還元**

といった感じ。

具体的なやり方は[効率的スモールハックさんの記事](https://small-hack.com/payment/)がとても分かりやすいです。
